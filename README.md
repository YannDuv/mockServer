# MockServer

## Install
Requires :
* nodeJs v4 or more
* grunt ==> `npm install -g grunt-cli`
* bower ==> `npm install -g bower`
* [mongoDB](https://docs.mongodb.com/manual/installation/)

Terminal :
```
npm install
bower install
```

## Run
```
cd my_path
grunt
```
