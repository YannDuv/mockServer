# MockServer

## Install
Requires :
* nodeJs v4 or more
* grunt ==> `npm install -g grunt-cli`
* bower ==> `npm install -g bower`

Terminal :
```
npm install -g json-server
npm install
bower install
```

## Run
```
cd my_path
grunt
```
